# 🏓 Juego de Ping Pong con Python

Un juego clásico de ping pong para dos jugadores, creado con Python y la librería Turtle.

## 🎮 Cómo jugar

- **Jugador izquierdo (rojo)**: Teclas **W** (arriba) y **S** (abajo)
- **Jugador derecho (azul)**: **Flecha arriba** y **Flecha abajo**

## 📋 Requisitos

- Python 3.x
- Turtle (viene instalado con Python)

## 🚀 Cómo ejecutarlo

1. Clona este repositorio
2. Ejecuta: `python ping_pong.py`
3. ¡Haz clic en la ventana y a jugar!

## ✨ Características

- Paletas de colores (rojo y azul)
- Pelota verde
- Marcador en tiempo real
- Velocidad ajustada para principiantes
